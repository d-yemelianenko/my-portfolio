This is my personal portfolio built with **CakePHP 4.5** and containerized using **Docker & Docker Compose**.

## Features
- CakePHP 4.5 application
- Dockerized Apache + PHP 8.2
- Optional MySQL support
- Professional setup using Composer for PHP dependencies
- Ready to run locally without database

## Requirements
- Docker & Docker Compose
- Composer (for PHP dependencies)

## How to run locally

1. Clone the repository:
```bash
git clone https://github.com/TWOJ_LOGIN/my_portfolio.git
cd my_portfolio/app
```

2.Build and start Docker containers:
```bash
docker compose build
docker compose up -d
```

3.Open in your browser:

Then visit `http://localhost:9001` to see the welcome page.
